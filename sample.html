Text resize

This code will allow developers to add image and text resizing to elements on their website. Users will be able to change the size of various elements in the website with ease.

Text size changing only works with the p tag but you can target a div and change the size of all p tags inside of it. So, you can target a div or individual p tags if you want.

The rules for images are similar to text.

To use this you will need some way to run the ElementSizeChanger function.
The ElementSizeChanger function takes three parameters: element_class_name, element_id_with_number_value, upper_lower_limit_array.

Example: ElementSizeChanger('text', 'TextChanger', [10, 900])

Multiple input boxes can control a single element so an override input is possible.
For instance, a user could remove all description text from an article list by changing the text size to 0 so nothing but the headlines show by using the override.

Then increase the individual articles they like using the text size changer next to it.

  • Images uses px
  • Text uses pt
